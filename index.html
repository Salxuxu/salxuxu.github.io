Focal Point Library Homepage Redesign

Transforming a static internal homepage into a dynamic dashboard for coaches.

Role
Lead Product Designer
Scope
UX Research · IA · UI Design
Outcome
+35% Event Engagement

The Problem

Understanding the Problem

📋
Lack of User Value
The homepage only displayed a welcome banner and phishing advisory, offering coaches no actionable resources.
🧭
Poor Navigation
Unclear entry points made it difficult for coaches to find what they needed, leading to abandonment.
📉
Low Engagement
The static nature didn't encourage repeat visits. Coaches couldn't see events or updates at a glance.

Research & Discovery

How I Found the Answers

🎙
User Interviews
Conducted conversations with coaches and stakeholders to identify priority needs and pain points.
🗺
Priority Mapping
Identified core resources coaches needed: events, announcements, newsletters, and training materials.
🏗
Information Architecture
Mapped categories and created a mental model aligned with how coaches naturally think about resources.
Key Insights
Coaches need quick access to upcoming events and training opportunities
Community recognition and wins drive motivation and engagement
Updates on new tools need to be surfaced immediately
A scannable, card-based layout enables faster information discovery

The Solution

A Dashboard That Works for Coaches

I redesigned the homepage into a functional dashboard that surfaces the most important resources coaches need, all in one scannable, card-based layout.

📢 Announcements

📅 Upcoming Events

📰 Newsletter Access

🎉

Wins & Library Updates

Community recognition and new tools drive repeat visits.

Impact & Results

Measurable Outcomes

10%
Traffic Increase
Post-launch library traffic growth
+35%
Event Engagement
More coaches attending events and trainings
↑
Repeat Visits
Coaches returning for announcements and updates

"Much more usable and relevant. I can find what I need without digging around."

Coach, Focal Point

Reflection

Key Learnings

01
Internal Tools Need Strong UX
Even internal tools benefit from thoughtful UX — making a measurable difference in engagement and adoption.
02
Content-Driven Dashboards Drive Engagement
Dynamic cards created an experience coaches wanted to return to regularly.
03
Qualitative Research Reveals Hidden Pain Points
User interviews surfaced the motivational role of community recognition.
04
Business & User Goals Can Align
Community, training, and events objectives aligned with user needs when designed thoughtfully.

FP Resource Center Redesign

Redesigning the tool hub for better clarity and consistency.

Overview

Bringing Clarity to Complexity

The FP Resource Center suffered from fragmented navigation and inconsistent visual design that made it hard for users to find what they needed.

The redesign established a clear visual hierarchy, consistent component language, and intuitive pathways — reducing cognitive load and improving task completion.

Key Improvements

What Changed & Why

🎨
Visual Consistency
Unified color, typography, and component styles — eliminating the patchwork feel of the legacy design.
🗂
Clearer Navigation
Reorganized information architecture based on user mental models.
⚡
Faster Workflows
Surfaced frequently used tools at the top level with quick-access patterns for power users.
